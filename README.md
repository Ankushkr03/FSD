# Full Stack Development (FSD) Repository

Welcome to the **FSD** (Full Stack Development) repository. This repository contains a collection of experiments, tasks, and testing modules covering both frontend and backend development.

## 📂 Project Structure

The repository is organized into the following main sections:

### 🧪 Experiments (Frontend)
These are React-based projects built using **Vite**.
- **Experiment-1 to Experiment-6**: Various frontend experiments exploring React components, hooks, and styling.
- **Task-2**: A specific frontend task implementation.

### ⚙️ Backend
Located in the `/Backend` directory, these experiments focus on server-side logic, APIs, and database integration.
- **Experiment-8, 9, 11, 13**: Python-based backend experiments (likely using Flask or similar frameworks).

### 🧪 Testing Module
The `/Testing` directory contains a complete full-stack setup designed for testing purposes:
- **Testing/Backend**: A Flask application with RESTful routes for student management.
- **Testing/Frontend**: A React/Vite application utilizing Material UI (MUI) for a clean user interface, integrated with the testing backend.

## 🚀 Getting Started

### Prerequisites
- **Node.js** (for frontend projects)
- **Python 3.x** (for backend projects)
- **npm** or **yarn**

### Running Frontend Experiments
1. Navigate to the desired experiment directory:
   ```bash
   cd Experiment-X
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Running the Testing Backend
1. Navigate to the testing backend directory:
   ```bash
   cd Testing/Backend
   ```
2. (Optional) Create and activate a virtual environment.
3. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python run.py
   ```
